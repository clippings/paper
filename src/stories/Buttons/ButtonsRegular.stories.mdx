## Stories

#### Primary

The Primary variant is only variant what is  going to be affected by the change of the theme

<Canvas>
  <Story
    name="Primary button big"
    args={{
      theme: 'light',
      variant: BUTTON_VARIANT.PRIMARY,
      size: BUTTON_SIZE.BIG

    {(args) => <Button
        endIcon={<Icon icon={ICON.BOARD} color={ICONS_COLOR.DARK} size={ICONS_SIZE.SMALL} />}

        Primary big
